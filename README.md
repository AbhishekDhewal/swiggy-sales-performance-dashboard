## Project Overview
This project analyzes Swiggy sales data using Microsoft Excel to identify key business insights related to revenue, order trends, and customer behavior.

The dashboard is designed to help stakeholders understand performance metrics and make data-driven decisions.

## Dataset
- Total Orders: 197,000+
- Data Source: Swiggy order-level transactional data
- Format: Excel (.xlsx)

## Key Analysis Performed
- Revenue and order trend analysis
- Top-performing cities and restaurants
- Customer ordering behavior
- Monthly and category-wise sales performance

## Tools & Technologies
- Microsoft Excel
- Pivot Tables
- Power Query
- Data Cleaning & Transformation
- Data Visualization  

## Key Insights
- Identified top revenue-generating cities and time periods
- Discovered seasonal demand patterns
- Highlighted high-performing restaurant categories
